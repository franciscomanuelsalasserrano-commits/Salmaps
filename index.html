Instalar TACNAV

Para aprovechar toda la pantalla y bloquear mejor el modo horizontal, instala esta web como aplicación.

  1. Android / Chrome: pulsa el menú ⋮ y elige Instalar app o Añadir a pantalla de inicio.
  2. iPhone / Safari: pulsa compartir y después Añadir a pantalla de inicio.

Una vez instalada, ábrela desde el icono del móvil para verla a pantalla completa.
